# Game of life

An alternative implementation using [pyxel](https://github.com/kitao/pyxel).

## TL;DR

```bash
pyxel package ./game_of_life ./game_of_life/ui.py
pyxel play game_of_life.pyxapp
pyxel app2html game_of_life.pyxapp
```
